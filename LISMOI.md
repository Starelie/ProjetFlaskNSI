# ProjetFlaskNSI

### Prérequis:

Pour que ce projet fonctionne, vous devez installer les modules Python nécessaires avec la commande suivante:
```
python3 -m pip install -Ur requirements.txt
```

Pour convertir des fichiers vidéo, vous devez télécharger [ffmpeg](https://ffmpeg.org/download.html) et l'ajouter à votre variable d'environnement PATH

Si vous souhaitez convertir des fichiers de documents, vous devez télécharger [pandoc](https://pandoc.org/installing.html) et à votre variable d'environnement PATH

### Comment exécuter:

Pour exécuter le serveur Flask, vous devez exécuter le fichier `flask_app.py` avec la commande suivante :
```
python3 flask_app.py
```

Ouvrez ensuite `http://127.0.0.1:5000` dans un navigateur web ou faites `ctrl + clic` sur le lien dans le terminal.
