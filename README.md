# ProjetFlaskNSI

### Requirements:

For this project to work, you need to install the necessary python modules with the following command:
```
python3 -m pip install -Ur requirements.txt
```

If you want to convert video files, you must download [ffmpeg](https://ffmpeg.org/download.html) and add it to PATH.

If you want to convert document files, you must download [pandoc](https://pandoc.org/installing.html) and add it to PATH.

### How to run:

To run the flask server, you must execute the `flask_app.py` file by running the following command:
```
python3 flask_app.py
```

Then open `http://127.0.0.1:5000` on a web browser or `ctrl + click` on the link in the terminal
